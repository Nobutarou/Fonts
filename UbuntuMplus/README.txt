Mplus font uses Mplus font license. Mplus font license doesn't prohibit to distribute this
font with another license.

Ubuntu font uses Ubuntu font license. Any mod font can not be published with another
license.

Then, this font use Ubuntu font license. Read LICENCE.txt for the detail.

でも Ubuntu フォントは Ubuntu font license じゃなきゃ再配布はだめと書いてあります。

なので、このフォントは Ubuntu Font License に従います。詳しくは LICENCE.txt 読んでください
